Esta aplicação foi desenvolvida como um projeto pessoal para facilitar a descoberta de filmes através de recomendações aleatórias diárias.
O objetivo é eliminar a minha indecisão na hora de escolher os filmes, oferecendo uma sugestão escolhida aleatóriamente baseada no catálogo do The Movie Database (TMDB).

A aplicação consome a API do TMDB para obter as informações atualizadas sobre os filmes. Inicialmente só está utilizando informações como título, poster e plataformas de streaming que disponibilizam o filme.
